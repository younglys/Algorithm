<h3>가운데 문자 출력</h3>
<p>소문자로 이루어진 입력된 문자열의 가운데 문자 출력</p>
<p>단어의 길이가 짝수일 경우 가운데 2개의 문자 출력</p>

```js
function solution(s) {
  let answer = "";
  let middle = s.length / 2;

  if ((s.length) % 2 === 1) {
    answer = s.split('')[Math.floor(middle)];
  } else {
    answer = s.slice((middle - 1), (middle + 1));
  }
  return answer;
}

let str = "algorithm";
let str2 = "struggling";
console.log(solution(str));
```
